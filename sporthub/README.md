# Sports Authority of Goa - Sporthub

Welcome to the Sports Authority of Goa (SAG) website project. This static website provides information about sports activities, athletes, grounds, and events in Goa.

## Project Structure

- **index.html**: The main landing page.
- **about.html**: Information about the SAG team and mission.
- **tradionalSp.html**: Details on traditional Goan sports like Lagori and Viti Dandu.
- **athelets.html**: Profiles of Goan athletes.
- **grounds.html**: Locations and details of sports grounds (Mapusa, Panaji, etc.).
- **upcoming.html**: Schedule of upcoming sporting events.
- **past.html**: Archive of past events and news.
- **css/**: Stylesheets (Bootstrap, custom styles).
- **js/**: JavaScript files (jQuery, Bootstrap, custom scripts).
- **images/**: Images used across the site.

## Technologies

- HTML5 & CSS3
- Bootstrap 4 Framework
- jQuery
- AOS (Animate On Scroll) Library

## How to Run

Simply open `index.html` in any modern web browser.
